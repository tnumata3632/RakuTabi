
# RakuTabi (楽旅)

**自分が知っている場所しか目的地が思い浮かばない**  
**急な休みが取れて旅行に行きたい人** 向けの  
**楽旅（仮）** という **ツアー推薦サービス** です。  

これは、~~日程と予算のみの~~**シンプルな条件でツアーを検索** でき、   
**楽天トラベルのようなWebサービスや旅行代理店のカウンター** とは違って、   
**エリアを指定せずに、思いつかないような場所や変わったプランを推薦する機能** が備わっています。
